## Οδηγίες Εγκατάστασης

Για να τρέξει το project σε περιβάλλον LAMP / XAMPP:

1. Κάνουμε clone ή κατεβάζουμε το repository στον φάκελο του server.
2. Τοποθετούμε το project μέσα στο `htdocs`.
3. Δημιουργούμε τη βάση δεδομένων από το `schema.sql`.
4. Κάνουμε import τα αρχικά δεδομένα από το `seed.sql`.
5. Ελέγχουμε το αρχείο `includes/db.php` ώστε τα στοιχεία σύνδεσης στη βάση να είναι σωστά.
6. Εκκινούμε Apache και MySQL από το XAMPP.
7. Ανοίγουμε το project από τον browser.
